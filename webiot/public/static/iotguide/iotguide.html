图解创趣物联数据通讯过程

1 准备

1.1 注册账号

浏览器访问http://106.54.66.50:8080（推荐谷歌浏览器，不支持低版本IE）

  • 点击注册账号

  • 填写注册信息

  • 邮箱查阅验证码

  • 完成注册

1.2 登录账号

2 管理用户/设备信息

2.1 用户 (⭐用户名①)

  • 点击头像，可查看/修改部分个人信息（更改头像，重命名，修改密码，权限升级）

2.2 设备

2.2.1 设备增删改查 （⭐设备id②）

  • 在菜单栏中，选择设备信息，这里会展示所有注册设备的相关信息

  • 点击管理设备，可新增、删除、修改设备

    注意： 新账号默认注册设备创趣小屋（不喜欢的话，自行修改），能够注册的设备数量与用户等级相关

2.2.2 连接密钥查看/刷新(⭐设备码③)

  • 点击连接密钥查看当前密钥，点击生成可刷新密钥

所有注册设备共用一个连接密钥

3.3 Web平台控制智能硬件

【Web平台-控制界面】

  • 激活正在使用的设备脚本

功能介绍：

发送会话 - 主题：CmsgW ; 用于发送字符串信息，受接收端限制，安全字符为英文、数字、下划线

按钮 - 主题：Cbtn ; 虚拟按钮按下时，发送数值1信息；否则，发送0

滑杆 - 主题: Cran ; 虚拟滑杆取值范围0-100， 滑动发送停留位置数值大小

（注意：所有发送动作频率均为2s, 智能硬件接收存在延迟为正常现象）
